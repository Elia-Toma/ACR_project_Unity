# Smart Logistics Simulation

This project implements a warehouse logistics scenario using **ROS 2** for high-level control logic and **Unity** for 3D simulation and visualization.

## Overview
The system simulates autonomous robots executing logistics tasks coordinated by a central ROS 2 manager node.

## Prerequisites
* **ROS 2 Humble** (running via Docker on Ubuntu 22.04).
* **Unity 6000.0.62f1 LTS**.
* **Unity Robotics Hub** packages (ROS-TCP-Connector).
